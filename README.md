# Lab 3 – Matrix & Vector System

This project implements a matrix and vector system using C#.

## OOP Concepts Applied
- Encapsulation
- Composition (Matrix contains Row objects)
- Aggregation (Repository classes store objects)
- Dependency (Calculators depend on DimensionValidator)

## Features
- Matrix addition with dimension validation
- Vector addition with size validation
- Console-based input and output

## How to Run
1. Open terminal in project folder
2. Run: dotnet run
3. Enter values when prompted
